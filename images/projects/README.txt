Each project has its own numbered folder.

Structure:
  01/
    cover.jpg   ← shown in the home grid
    01.jpg      ← hero image on the project page (large)
    02.jpg      ← grid image
    03.jpg      ← grid image
    ...up to 09.jpg

  02/
    cover.jpg
    01.jpg
    ...

Folders match the project order in index.html (SITE.projects).
Missing images are silently skipped — no code changes needed.
Supported formats: jpg, png, gif, webp
